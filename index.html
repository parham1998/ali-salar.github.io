Implementation of the Pentago game using the mini-max algorithm and alpha-beta pruning

Ali Salar & Erfan Momeni

Languages: Java and JavaFX

Pentago is a strategy board game designed for two players. You can also play with AI.
In order to implement this game in Java, we used a combination of algorithms such as the mini-max tree and alpha-beta pruning to help the computer make informed decisions about its next move.
The mini-max tree algorithm is a decision-making algorithm that explores all possible outcomes of a game by generating a tree of all possible moves, assuming that each player makes the best possible decision at each turn. Alpha-beta pruning is an optimization technique used in the mini-max tree algorithm to eliminate branches of the tree that are unlikely to lead to a winning outcome.
The game's user interface was implemented using JavaFX, a Java library used for building desktop applications with rich graphical user interfaces.

Implementation of Neural Style Transfer technique, drawing inspiration from the paper titled "Image Style Transfer Using Convolutional Neural Networks"

Languages: PyTorch

Neural style transfer (NST) is a popular optimization technique used in the field of computer vision that involves creating a new image by combining the style of one image and the content of another. This process involves taking two input images: a content image and a style reference image. The content image typically represents the subject or object that the user wishes to stylize, while the style reference image represents the particular style that the user wishes to apply to the content image. The style reference image is usually an artwork created by a famous painter or designer, and it can be used to impart specific visual characteristics, such as texture, color, or composition, to the final stylized image.

Implementation of a pre-trained CNN for the covid-19 recognition from chest X-Ray images

Languages: PyTorch

The process of diagnosing diseases typically involves various medical procedures and examinations. In the case of diagnosing COVID-19, one way to identify the disease is through a chest scan. This involves taking a photograph of the chest, which allows doctors to carefully monitor the health of the lungs. However, to accurately diagnose COVID-19 using deep learning techniques, we need to have a dataset that contains relevant data.
The dataset we will be using for training our deep learning model includes Corona-positive CT scan data collected by Joseph Cohen, Ph.D., from the University of Montreal. This data was gathered specifically to aid in the diagnosis of COVID-19 and will be a critical component of our model's training. In addition to the Corona-positive data, we will also use the Chest X-Ray Images database from the Kaggle site to provide Corona-negative or normal samples. This dataset will provide a baseline for comparison and help our model differentiate between healthy and infected lungs.
Once we have gathered both the Corona-positive and Corona-negative datasets, we will be able to build a comprehensive database that can be used to train our deep learning network. This database will be the foundation for our model, allowing it to learn from the different examples and accurately diagnose COVID-19. Ultimately, the goal is to create a model that can accurately diagnose COVID-19 from a chest scan, which can help medical professionals make informed decisions about treatment options and patient care.

Implementation of “short-time autocorrelation”, “short-time AMDF”, and “real cepstrum” for speech pitch detection

Languages: Matlab

Short-time autocorrelation, short-time AMDF (average magnitude difference function), and real cepstrum are commonly used signal processing techniques for detecting pitch or fundamental frequency in speech signals.
Short-time autocorrelation is an effective pitch detection method for speech in noisy environments, involving computing autocorrelation of a small speech segment and identifying the fundamental frequency from the maximum autocorrelation lag.
The short-time AMDF method for speech pitch detection calculates the average absolute difference between overlapping segments and estimates the fundamental frequency as the time lag that minimizes the average difference, being computationally efficient and suitable for real-time applications.
The real cepstrum method for speech pitch detection involves the inverse Fourier transform of the logarithm of the magnitude of the Fourier transform to identify the fundamental frequency as the peak in the cepstral representation, being robust to noise and non-stationary signals but requiring more computational resources than the other methods.

Implementation of a two-layer perceptron (from scratch) with four back-propagation methods to estimate the Mackey-Glass time series

Languages: Python

back-propagation methods:
1. stochastic gradient descent (SGD): this method updates the network parameters (weights) at every time step, making it sensitive to noisy data.
2. emotional learning: this method is very similar to the SGD method, but uses the previous step error to make the network learn faster and more accurately.
3. adaptive learning rate: in this method, we assign a different learning rate to each of the trainable parameters, which are called adaptive learning rates. For each element of the weights matrix, we consider a different learning rate, which is also trained during the learning process.
4. levenberg marquardt: This algorithm is a generalization of the Gauss-Newton algorithm, designed to increase the convergence speed of the second-order optimization. We can adjust its parameters to behave like Gauss-Newton or SGD. Unlike the aforementioned methods, we do not update the parameters after each new training sample is received. Instead, the parameters are updated after all training samples have been received. After each epoch, parameters are updated based on the jacobian matrix, which stores all epoch information. This method is suitable for small datasets and its computational costs increase significantly with larger datasets.

Implementation of knight’s tour problem using Warnsdorff's rule

Languages: Java and JavaFX

The knight's tour problem is a well-known problem in the field of chess, where the objective is to find a sequence of moves for a knight to visit every square on a chessboard exactly once. Warnsdorff's rule is a heuristic algorithm that can be used to solve the knight's tour problem.
The idea behind Warnsdorff's rule is that the knight should always move to the square with the fewest options available. This reduces the chance of getting stuck and increases the chance of finding a complete tour. The rule is not foolproof, and it may not always find a solution, but it has been shown to be effective in most cases.
